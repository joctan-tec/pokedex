# Pokedex
This Pokédex app lets users explore and search for Pokémon by fetching real-time data from an external API. It's a lightweight and interactive tool for any Pokémon fan.

## User Instructions

La aplicación cuenta con la siguiente pantalla:

<img width="1917" height="913" alt="image" src="https://github.com/user-attachments/assets/765e0a7d-ca21-41d4-a888-53453dfff942" />

Donde el usuario puede realizar las siguientes acciones: 

1. Al ingresar, podrá ver un pokemón desplegado en la sección de Información, con datos relevantes como nombre, altura, peso, especie, fotografia, cadena de evolución, etc.
   <img width="1917" height="913" alt="image" src="https://github.com/user-attachments/assets/349a6e6e-7b69-49cf-aa95-d03a6d212986" />
2. También tendrá a la izquierda de la pantalla, un menú deslizable donde podrá seleccionar cualquier pokemón que desee para ver su información.
   <img width="1919" height="912" alt="image" src="https://github.com/user-attachments/assets/53e1eb4c-de2d-429d-abbe-9abf5ad0ca0f" />
3. En el menú deslizable, podrá ir hasta el final y le cargarán nuevos pokemons conforme vaya bajando. Se cargarán de 20 a la vez hasta llegar al final.
   <img width="1916" height="912" alt="image" src="https://github.com/user-attachments/assets/4d114a7e-f8b0-4ce7-b6e1-dde98c8cb155" />
4. En el menú, puede seleccionar cualquier pokemon y al seleccionarlo, se desplegará en la seccion de información.
